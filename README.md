# Parcmall - Coming Soon Page

## 🏪 Giới thiệu
Trang web "Coming Soon" chính thức cho **Parcmall** - trung tâm mua sắm hiện đại và tiện nghi nhất sắp ra mắt.

## ✨ Tính năng
- **Countdown Timer**: Bộ đếm ngược 30 ngày tự động
- **Responsive Design**: Tương thích với mọi thiết bị
- **Modern UI**: Giao diện đẹp mắt với Tailwind CSS
- **Social Media Links**: Liên kết trực tiếp đến Facebook và Instagram
- **Custom Banner**: Ảnh banner tùy chỉnh cho Parcmall

## 🚀 Cách chạy
1. Clone repository này
2. Mở file `index.html` trong trình duyệt
3. Hoặc sử dụng Live Server để chạy local

## 🎨 Công nghệ sử dụng
- **HTML5**
- **Tailwind CSS** (CDN)
- **JavaScript** (Vanilla)
- **Font Awesome** (Icons)
- **Google Fonts** (Typography)

## 📱 Responsive
- Desktop: Giao diện đầy đủ với layout 2 cột
- Mobile: Tối ưu cho màn hình nhỏ
- Tablet: Tương thích với iPad và tablet

## 📅 Countdown
Bộ đếm ngược tự động 30 ngày từ thời điểm hiện tại, hiển thị:
- Ngày (Days)
- Giờ (Hours) 
- Phút (Minutes)
- Giây (Seconds)
